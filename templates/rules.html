The game of Boggle is a classic word game played on a square board with a number of dice. Each die has a letter on each face. The dice are rolled each round to produce a random board.

The game is begun by deciding the final score needed to the win the game. This is entirely up to the players. A common goal is 50-100 points.

Once a winning score is decided, all the dice are rolled and the round begins when the timer starts. The exact allotted time depends on the variant of the game being played, but it is usually between 3-5 minutes, though the time can also be adjusted according to the player's preference.

After the board is rolled, the timer is set and the players must write down each word they find on the board. Words can be made of any adjacent dice (even diagonal), but a die cannot be repeated in a single word. Keep in mind that the dictionary to be used for allowed words should also be decided before the game begins.

After time is up, the players will all compare notes on which words were found. If two or more players found the same word (even if the path for the found word is different) all players mark that word off their lists. Any surviving words in a player's list are summed up according to the point table below to calculate the player's score for each round.

Each word is assigned a point value according to it's length.
